## **📌 业务流程（串联所有模块）**

我会设计一条完整的 **业务链路**，让所有模块 **自然关联**，确保数据流畅。

### **🛠️ 1. 登录 & 身份验证**

1. **工人 / 经理 / 管理员** 登录系统
2. **记录登录信息**（IP、设备、时间）
3. **跳转到公告页面**，查看最新公告

🔗 **关联模块：**
 ✅ **登录** → **登录信息管理**（记录日志）
 ✅ **登录成功后，查看公告**（公告管理）

------

### **📢 2. 公告管理**

1. **管理员 / 经理发布公告**（如安全提示、项目进展）
2. **工人 / 经理查看公告**
3. **公告按用户角色 & 部门推送**（避免无关人员收到）

🔗 **关联模块：**
 ✅ **公告管理** → **部门管理**（公告可按部门发布）
 ✅ **公告管理** → **人员管理**（公告只发给相关人员）

------

### **🏗️ 3. 项目管理**

1. **管理员 / 经理 创建施工项目**
2. **分配工人到项目**
3. **施工任务由经理分配 & 进入工作流审批**
4. **工人开始施工，更新进度**
5. **完工后，经理确认**

🔗 **关联模块：**
 ✅ **项目管理** → **人员管理**（分配工人）
 ✅ **项目管理** → **任务管理（工作流）**（工人任务审批 & 进度更新）

------

### **📝 4. 任务管理（工作流）**

1. **经理给工人分配任务**
2. **任务进入“进行中”状态**，工人开始施工
3. **工人反馈进度，经理审核**
4. **任务完成后，经理确认**

🔗 **关联模块：**
 ✅ **任务管理（工作流）** → **项目管理**（施工进度更新）
 ✅ **任务管理（工作流）** → **公告管理**（完工后发布公告）

------

### **💼 5. 请假管理（工作流）**

1. **工人提交请假申请**
2. **经理审批**（批准 / 拒绝）
3. **请假通过后，工人考勤更新**
4. **影响施工任务，经理需重新调整排班**

🔗 **关联模块：**
 ✅ **请假管理（工作流）** → **人员管理**（请假影响工人状态）
 ✅ **请假管理（工作流）** → **任务管理**（任务需重新分配）

------

### **🔧 6. 后勤管理**

1. **工人申请住宿 / 领用设备 / 领取物资**
2. **管理员分配房间 / 设备**
3. **工人开始使用，并记录使用情况**
4. **施工结束后，归还物资**

🔗 **关联模块：**
 ✅ **后勤管理** → **人员管理**（住宿 & 物资关联工人）
 ✅ **后勤管理** → **项目管理**（施工设备影响工期）

------

### **📊 7. 数据统计 & 监控**

1. **管理员查看人员分布 / 项目进度 / 物资使用情况**
2. **查看各项目施工情况**（延误 / 进展顺利）
